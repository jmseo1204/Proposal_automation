
[상황] 
너는 과거 프로젝트의 기록을 중요한 내용만 뽑아서 요약하는 일을 해야해. 이때 과거 프로젝트 기록에는 기업명, 프로젝트 주제, 진행 프로세스를 포함하고 있어. 
너는 여기서 각 주제가 어떤 프로세스로 다루어졌는지 이하 예시처럼 작성해주면 돼.

[형식]
기업명

프로젝트 주제1
- 해당 주제를 어떻게 접근했는지
프로젝트 주제2
- 해당 주제를 어떻게 접근했는지
프로젝트 주제3
- 해당 주제를 어떻게 접근했는지

[예시]
리코

1. 폐기물 수거 최적 스케줄링 모델 구현
- OR-Tools와 Genetic Algorithm을 활용한 최적화 알고리즘 제작

2. 배차 스케줄 평가 지표 탐색
- 클러스터링 알고리즘을 통한 배차 경로 최적화 및 분석


[작성 조건]
불릿포인트를 무조건 3개 이내로 활용해야해. 
EDA, 배경지식 탐색, 선행 조사같은 일반적으로 진행하는 단계는 포함하지 말고, 해당 주제에만 적용되는 특별하고 전문적인 내용으로 불릿포인트를 구성해줘.
